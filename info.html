To-Do App Information

Overview

This To-Do App is designed to help users efficiently manage their tasks with AI-powered assistance.

Features

  • AI-powered task assistance and prioritization
  • Task summarization and description enhancement
  • Smart category suggestions for tasks
  • Secure API with rate limiting to prevent misuse

Storage

Tasks are stored in a JSON file, allowing easy retrieval, updates, and persistence.

Technologies Used

This app is built using:

Flask - Python web framework
Semantic UI - Frontend design
Google Gemini AI API - AI-powered task assistance
JSON - Simple and efficient data storage

How to Use

Navigate to the homepage to:

  • Add new tasks
  • Update existing tasks
  • Delete completed tasks
  • Use AI for prioritization, summarization, and categorization
